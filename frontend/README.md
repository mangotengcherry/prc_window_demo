# Frontend

Vue 3 + Vite + TypeScript workbench frontend using Pinia, Vue Router, Element Plus, and Plotly.

## Run

```bash
npm install
npm run dev
```

Set `BACKEND_URL` when the FastAPI backend is not on `http://localhost:8000`:

```bash
BACKEND_URL=http://127.0.0.1:8001 npm run dev
```

## Build And Test

```bash
npm test
npm run build
```
